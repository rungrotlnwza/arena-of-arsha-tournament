# DevTeam Framework 2

DevTeam Framework 2 คือเฟรมเวิร์กสำหรับพัฒนาเว็บด้วย Node.js (Express + EJS Layouts)
ออกแบบมาเพื่อรองรับการพัฒนาเว็บแบบ Single Port โดยรวมทั้ง Frontend และ Backend ไว้ในเซิร์ฟเวอร์เดียว

---

## Installation

### Requirements

- Node.js v16 ขึ้นไป
- npm

### Install

```bash
npm install
```

---

## Running the Project

### Development Mode

```bash
npm run dev
```

เซิร์ฟเวอร์จะรันที่:

```
http://localhost:3000
```
