# คู่มือการใช้งานโปรเจกต์

เอกสารในโฟลเดอร์นี้สรุปวิธีใช้งานระบบสำหรับ 2 กลุ่มหลัก

- ผู้ใช้งานทั่วไป: ดูหน้าเว็บ, สมัครทีม, ติดตามรายชื่อทีม, ดูตารางแข่ง, ดูหน้าถ่ายทอดสด
- ผู้ดูแลระบบ: ล็อกอิน, ตั้งค่าการแข่งขัน, ตรวจทีม, จัดการตารางแข่ง

ไฟล์ในชุดเอกสารนี้

- `README.md` : ภาพรวมเอกสารและลำดับการใช้งาน
- `user-guide.md` : คู่มือสำหรับผู้ใช้งานทั่วไป
- `admin-guide.md` : คู่มือสำหรับผู้ดูแลระบบ
- `images/` : โฟลเดอร์เก็บภาพประกอบเอกสาร

## ลำดับการใช้งานที่แนะนำ

### ฝั่งผู้ใช้งานทั่วไป

1. อ่านรายละเอียดกิจกรรมจากหน้าแรกและหน้ากติกา
2. เข้าหน้าสมัครและกรอกข้อมูลทีม
3. ติดตามสถานะผ่านหน้ารายชื่อทีม
4. รอดูสายการแข่งที่หน้า bracket
5. ติดตามถ่ายทอดสดในวันแข่ง

### ฝั่งแอดมิน

1. ล็อกอินที่ `/admin/login`
2. ตรวจหน้าสรุปที่ `/admin/dashboard`
3. ตั้งค่ากิจกรรมและช่วงรับสมัครที่ `/admin/settings`
4. ตรวจทีมและอนุมัติ/ปฏิเสธที่ `/admin/teams`
5. จัดการ bracket ที่ `/admin/bracket`

## หมายเหตุสำคัญ

- หน้าแอดมินใช้ JWT cookie สำหรับยืนยันตัวตน
- ถ้า token หมดอายุหรือไม่มีสิทธิ์ ระบบจะพากลับไปหน้า `/admin/login`
- การรีเซ็ต bracket จะล้างการจับคู่ทั้งหมด แต่ไม่ลบทีมสมัคร

## การใส่รูปในเอกสาร

สามารถใส่รูปในเอกสาร Markdown ได้เลย โดยแนะนำให้เก็บไฟล์ไว้ในโฟลเดอร์ `docs/project/images`

ตัวอย่างการอ้างรูป

```md
![หน้าสมัคร](./images/user-register.png)
![หน้าแอดมินบรากเก็ต](./images/admin-bracket.png)
```

คำแนะนำ

- ใช้ชื่อไฟล์สั้น ๆ อ่านง่าย
- แนะนำให้ใช้ตัวพิมพ์เล็กและคั่นคำด้วย `-`
- ถ้าเป็น screenshot ของ user/admin ให้ตั้งชื่อแยกกันชัดเจน
