# คู่มือผู้ใช้งานทั่วไป

เอกสารนี้อธิบายการใช้งานหน้าเว็บสำหรับผู้สมัครและผู้ติดตามการแข่งขัน

## หน้าหลักที่ใช้งานได้

- `/` : หน้าแรกของรายการแข่งขัน
- `/rules` : หน้ากติกาการแข่งขัน
- `/register` : หน้าสมัครทีม
- `/register-success` : หน้ายืนยันหลังสมัครสำเร็จ
- `/teams` : หน้ารายชื่อทีมที่ได้รับการอนุมัติ
- `/bracket` : หน้าตารางการแข่งขัน
- `/live` : หน้าถ่ายทอดสด

## 1. การสมัครทีม

หน้า `/register` ใช้สำหรับส่งข้อมูลทีมเข้าแข่งขัน โดยระบบจะโหลดข้อมูลกิจกรรมจากการตั้งค่าปัจจุบันก่อน

ข้อมูลที่ต้องกรอก

- ชื่อทีม
- ข้อมูลผู้เล่น 2 คน
- `Family Name`
- ชื่อตัวละครในเกม
- `Discord ID`

ข้อมูลเพิ่มเติมที่กรอกได้

- ประสบการณ์การแข่งขัน
- ช่องทางที่รู้จักกิจกรรม
- ยินยอมให้ถ่ายทอดสด

เงื่อนไขสำคัญ

- ต้องมีผู้เล่น 2 คนเสมอ
- ต้องยอมรับกติกาการแข่งขัน
- ต้องยินยอมให้เก็บข้อมูลสำหรับการติดต่อ
- หากระบบปิดรับสมัครอยู่ จะไม่สามารถส่งฟอร์มได้

## 2. การแจ้งทีมสำรอง

ถ้าจำนวนทีมหลักครบตาม `max_teams` แล้ว ระบบยังรับสมัครต่อได้ แต่จะแจ้งว่าเป็นทีมสำรอง

พฤติกรรมของระบบ

- ก่อนกดสมัคร หน้าเว็บจะแสดงข้อความเตือนว่าถ้าสมัครตอนนี้จะเป็นทีมสำรองลำดับที่เท่าไร
- หลังสมัครสำเร็จ หน้า `/register-success` จะแสดงลำดับทีมสำรองอีกครั้ง

## 3. หลังสมัครสำเร็จ

เมื่อสมัครผ่าน ระบบจะพาไปหน้า `/register-success`

สิ่งที่ผู้ใช้จะเห็น

- ชื่อทีมที่เพิ่งสมัคร
- ข้อความยืนยันการสมัคร
- ข้อความแจ้งทีมสำรองถ้ามี
- ปุ่มไปยัง Discord ของกิจกรรม

คำแนะนำ

- ให้ติดตามข่าวสารผ่าน Discord เป็นหลัก
- รอตรวจสอบสถานะทีมจากทีมงาน
- เมื่อทีมได้รับอนุมัติแล้ว รายชื่อจะไปแสดงที่หน้า `/teams`

## 4. การดูรายชื่อทีม

หน้า `/teams` จะแสดงเฉพาะทีมที่ได้รับการอนุมัติแล้ว

ข้อมูลที่จะแสดงในแต่ละทีม

- ชื่อทีม
- ชื่อผู้เล่นจาก `Family Name`
- ชื่อตัวละครในเกม
- `Discord ID`

ถ้ายังไม่มีทีมที่ผ่านการอนุมัติ ระบบจะแสดงหน้า empty state แทนรายการทีม

## 5. การดูตารางการแข่งขัน

หน้า `/bracket` ใช้ติดตามสายการแข่ง

พฤติกรรมของหน้า

- ถ้ายังไม่มีการจับคู่ ระบบจะแสดง bracket เปล่าตามขนาดรายการ
- ถ้ามีการจับคู่แล้ว จะแสดงทีม, สถานะ, เวลาแข่ง และลิงก์สตรีมของแต่ละคู่ตามข้อมูลจริง
- ผู้ใช้สามารถลากเมาส์เพื่อเลื่อนดู bracket ขนาดใหญ่ได้

สิ่งที่ผู้ใช้จะเห็นในแต่ละคู่

- คู่ที่เท่าไร
- ทีมฝั่งซ้ายและขวา
- สถานะ เช่น รอแข่ง, กำลังแข่ง, จบแล้ว
- ข้อมูลเวลาแข่งขันถ้ามี
- ลิงก์ถ่ายทอดสดถ้ามี

## 6. การดูถ่ายทอดสด

หน้า `/live` ใช้ติดตามการถ่ายทอดสดของกิจกรรม

ระบบรองรับ 3 กรณี

- ถ้าใส่ลิงก์ Twitch จะฝัง player ให้ดูในหน้าได้เลย
- ถ้าใส่ลิงก์ YouTube จะฝัง player ให้ดูในหน้าได้เลย
- ถ้าเป็นลิงก์อื่น ระบบจะแสดงปุ่มให้กดออกไปดูภายนอก

ถ้ายังไม่มีลิงก์สตรีม ระบบจะแจ้งว่ายังไม่มีการถ่ายทอดสด

## 7. ปัญหาที่พบบ่อย

### สมัครไม่ได้

ตรวจสอบดังนี้

- เปิดรับสมัครอยู่หรือไม่
- กรอกข้อมูลผู้เล่นครบทั้ง 2 คนหรือไม่
- ชื่อทีมซ้ำกับทีมอื่นหรือไม่
- ยอมรับกติกาและยินยอมเก็บข้อมูลแล้วหรือยัง

### สมัครแล้วแต่ยังไม่ขึ้นหน้า `/teams`

เป็นพฤติกรรมปกติ เพราะหน้า `/teams` แสดงเฉพาะทีมที่ได้รับการอนุมัติแล้ว

### หน้า bracket ยังไม่มีคู่

เป็นพฤติกรรมปกติถ้าทีมงานยังไม่เริ่มจับคู่หรือยังไม่ได้สร้างตารางแข่งขัน
